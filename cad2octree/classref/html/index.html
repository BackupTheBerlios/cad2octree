cad2octree

Implementierung zur Diplomarbeit "Erzeugung und Evaluierung von Oktalbaumstrukturen als Schnittstelle zu CAD-Programmen" Diplomarbeit-Nr. 2035

Mit Hilfe dieses Programms können Geometrieinformationen aus gültigen DXF (Drawable Exchange Format) - Dateien extrahiert werden. Diese Daten werden zur Erzeugung einer Octree-Struktur verwendet.

  • Autor: Stefan Mahler
  • Betreuer: Ralf-Peter Mundani
  • erstellt: 2002
    Abteilung Simulation großer Systeme, Institut für Informatik, Universität Stuttgart
